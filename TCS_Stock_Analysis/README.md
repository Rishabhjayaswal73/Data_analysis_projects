#  TCS Stock Analysis & Trading Strategy

##  Overview
This project analyzes TCS stock data and builds a trading strategy using moving averages.

##  Tools Used
- Python (Pandas, NumPy)
- Matplotlib
- Machine Learning (optional)

##  Features
- Data Cleaning
- EDA (Exploratory Data Analysis)
- Moving Average Strategy (MA20 & MA50)
- Buy/Sell Signal Generation
- Profit Calculation

##  Result
- Total Profit: ₹960 (approx)

##  Conclusion
The moving average strategy shows how trends can be used to generate trading signals and evaluate profitability.
